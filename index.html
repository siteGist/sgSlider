sgSlider - simple jQuery slideshow plugin

Demonstration

sgSlider
seriously great slider

  • Checks if images had been uploaded
  • Reverse pixels/item counting
  • Has callback for current item
  • Vertical and Horisontal views
  • Cilcular movment animation
  • Custom animation steps
  • Animation in end
  • Diferent callbaks
  • Easy styling

First we build an HTML structure, or simply the <tag/> container 

<div id="slider">
	<div>
	   <ul> <!-- Or in this case you can use <ol/> -->
			<li>
				The first Slide Content
			</li>
			<li>
				The second Slide Content
			</li>
			<li>
				...
			</li>
	   </ul>
	</div>
</div>
We have another option, more simple wrapping: 
<div id="slider">
	<li>
		The first Slide Content
	</li>
	<div> <!-- We can use div`s also ... -->
		The second Slide Content
	</div>
	<article> <!-- ... and other alowed in structure tags -->
		...
	</article>
</div>

If we are sutisfied with item view, then we can skip this step. Otherwise ... 

<!-- we give some styles into -->
<style type="text/css" >
	/* style here */
<style>
<!-- ... or into css file included via -->
<link rel="stylesheet" href="Your_File_Name.css" />
For example you can use such styles. Don`t give styles to wrapper or to item holder, other way it will break the plugin work - if you dont do it carefully. 
#slider {
	/* here we paste the style fore hole slider ontainer */
}
	#slider .item /* we can change this class in plugin parameters */ {
		/* styles for items */
	}

Then we inculde, if not included, jQuery library into <head/> section:

<script type="text/javascript" src="jquery-1.7.1.min.js" ></script>
And of course the sgSlider plugin itself, with plugin styles 
<link type="text/css" rel="stylesheet" href="jquery.sgSlider_0.1.css" />
<script type="text/javascript" src="jquery.sgSlider_0.1.js"></script>

Finally the javascript code itself: 

$(document).ready(function() {
	$('#slider').sgSlider(); // if first murkup variant
	$('#slider').sgSlider({ buildHTML: true }); // if second markup variant
});

The Options - click to expand

Layout options

  • Change to true if you want to build the 2 wrapp div`s
    buildHTML: false,
  • Css selector, for specific items to be placed in slider. (Only for buildHTML: true)
    wrapSelector: '',
  • carcass:
    Variable classes for each wrapp level
    {
    • Class for the main element
      container: 'container',
    • The wrapper class, this obj wrapps the view.
      wrapper: 'wrapper',
    • This class is for item holder
      itemHolder: 'item-holder',
    • And finaly class for each item in the item holder.
      item: 'item'
    },
  • navigation:
    The next/prev navigation buttons properties
    {
    • You want to use navigation buttons?
      use: true,
    • You can give the specific class for "Go to Next" button
      classNext: 'next',
    • You can give the specific class for "Go to Prev" button
      classPrev: 'previous',
    • text:
      Change the buttons content, simple text or HTML markup for...
      {
      • "go to Previous" button, or/and for...
        prev: '‹',
      • ... "go to Next" button
        next: '›'.
      },
    • What "bind-type" events bind to those buttons
      bind: 'click'
    },

Slider behaviour options

  • If vertical position, check if height is previously setted, otherway it wil be zero-height
    vertical: false,
  • If next will be equal to current positio minus step
    reverse: false,
  • carousell:
    The carousell/circular options (must have more idias for this section :) ):
    {
    • Enable it?
      enable: false,
    • Animate it to one direction (set oposit with reverse: true)
      oneDirection: false
    },
  • The most important part for navigation buttons: 'full', 'N' items or 'Npx' where N is number.
    step: 'full',
  • If there a current item, how to align it? - 'left'='top', 'center', 'right'='bottom'
    align: 'center',
  • Where we start? - 'N' items or 'Npx' where N is number.
    startPosition: '0px',
  • currOnHover:
    Make hovered item current after timeout
    {
    • Eneble this?
      enable: false,
    • Timeout in miliseconds
      timeout: 500
    },
  • scrollOnHover:
    Scroll the slider while hovering above
    {
    • Enable this feature?
      enable: false,
    • If there is a current item, return after returnToCurrentTimeout ms?
      returnToCurrent: true,
    • Time in ms after we return to current item position, with right align option
      returnToCurrentTimeout: 1000,
    • Give some borders relative first/last slade scroll size, for "nonscrolling" to border limits
      sizeIndent: 0.5,
    • For non jumping, specify in percent relation to item holder size, else it will use an animate function
      maxJump: 0.1,
    • Time of animation if we exceded the item maxJump value (nead to resentense this comment... :()
      offsetAnimateTime: 500
    },
  • Animate if animation ends?
    animateWhenShowEnds: false,
  • The time of main animation (when goin to next/prev or to specific item)
    speed: 1200,
  • And easing effect for main animation. (will be nice to integrate it to scrollOnHover animation)
    easing: 'linear',
  • autoNext:
    Do some automaticks...
    {
    • Enabling auto sider movements
      enable: false,
    • Path to do auto movements: ['+Npx', 'Npx', N,-N,...] or to disable set to null
      way: null,
    • Repeat the auto movemnts: true, false, (int)N
      repeat: true,
    • Timeout when next movment will be triggered: (int)N
      timeout: 3475,
    • delay:
      Delay options after listed action was triggered inside plugin
      {
      • Delay stops the autoNext, and start again after multiplied timeout
        multiply: 3,
      • This actions enabelign delay...
        onAction: 'anext aprev'
      }
    },

Callback functions

  • When the sliders started. Best use for hidding the slider and show it smothly, for non-jumping page layout
    onStart: function() {},
  • When the current elements is specified. obj - the jQuery object of this item.
    onCurr: function(obj) {},
  • Function when main animation starts.
    onAniStart: function() {},
  • Function when main animation ends.
    onAniEnd: function() {}

Some samples what we used in demonstration section

$(document).ready(function() {
	$('#sliderDiferentHWBottom').sgSlider({
		step: '2',
		navigation: { use: true },
		autoNext: {
			enable: true,
			way: [1,'+4','1300px',-7], // Sign '+' does`n go as string if asigned as int. So we forced the string type.
			repeat: 7
		},
		animateWhenShowEnds: true,
		currOnHover: {enable: true},
		onStart: function() {
			$('#sliderDiferentHWBottom').css({overflow: 'visible', visibility: 'visible'}).fadeTo(0,0).fadeTo(3000,1);
		}
	});
	$('#sliderDiferentHWTop').sgSlider({
		navigation: { use: false },
		autoNext: {
			enable: true
		},
		scrollOnHover: {enable: true, maxJump: 0.3},
		onStart: function() {
			$('#sliderDiferentHWTop').css({overflow: 'visible', visibility: 'visible'}).fadeTo(0,0).fadeTo(3000,1);
		}
	});
	$('#sliderVerticalWide').sgSlider({
		vertical: true,
		reverse: true,
		autoNext: {
			enable: true
		},
		navigation: { text: {
			prev: 'scroll Top',
			next: 'scroll Down'
		} },
		onStart: function() {
			$('#sliderVerticalWide').css({overflow: 'visible', visibility: 'visible'}).fadeTo(0,0).fadeTo(3000,1);
		}
	});
	$('#sliderVerticalWide2').sgSlider({
		vertical: true,
		step: '2',
		autoNext: {
			enable: true
		},
		animateWhenShowEnds: true,
		carousell: {
			enable: true
		},
		navigation: { text: {
			prev: 'scroll Top',
			next: 'scroll Down'
		} },
		onStart: function() {
			$('#sliderVerticalWide2').css({overflow: 'visible', visibility: 'visible'}).fadeTo(0,0).fadeTo(3000,1);
		}
	});
});

#sliderBestWay {
	/* Of course for this method we must set the max transerve size of an item*/
	height: 100px;
	/*
	in case vertical: true - set width: Npx,
	and set actual height for vertical slider size, otherwize it will be height = 0 :)
	*/
	overflow: hidden;
}
	#sliderBestWay.container { 
		/* Must be so, because we wont see the prev/next buttons */
		overflow: visible;
	}
	#sliderBestWay ul,
	#sliderBestWay li {
		padding: 0px;
		margin: 0px;
		list-style: none;
	}
	/* and aproximate view will be like this*/
		#sliderBestWay ul {
			width: 200%;
			/* in case vertical: true - set height: 200% */
		}
		#sliderBestWay li {
			float: left;
		}
And anabeling the sgSlider 
$(document).ready(function() {
    $('#sliderBestWay').sgSlider();
});

				overflow: 'visible',
				visibility: 'visible'
